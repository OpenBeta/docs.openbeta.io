# Introduction

OpenBeta is an open source project dedicated to ensuring free and open access to rock climbing data in computer programming, research, and data science endeavors.

### What We Do

- Publish rock climbing datasets, volumes of crowd-sourced climbing information, created and shared by climbers, under an **open license**.
- Develop and maintain open source tools to enable the community to work with climbing data.

:::info Did you know?
OpenBeta is a 501(c)(3) nonprofit collective under the Open Collective Foundation. All donations are tax-deductible where applicable.
:::
